# LinkedIn Post Draft

I recently finished my final degree project for the Bachelor's Degree in Data Science at Universidad Siglo 21, Argentina.

The thesis focused on a prescriptive analytics system for personal financial health in a Swiss neobank context. Instead of leaving the work as a purely theoretical proposal, I decided to turn it into a functional portfolio project.

The result is Swiss Financial Health Advisor: a prototype that transforms synthetic Open Banking-style transaction data into:

- customer behavioral segments
- an explainable Financial Health Score
- personalized recommendations
- an interactive Streamlit dashboard

The project combines data science, fintech product thinking, explainability, and privacy-aware design. That combination matters especially in European and Swiss financial environments, where analytics systems need to be useful, auditable, and trustworthy.

What I built:

- synthetic banking transaction generator
- monthly feature engineering pipeline
- K-Means segmentation
- composite financial health scoring model
- rule-based prescriptive recommendation engine
- Streamlit demo dashboard

This project helped me connect academic research with implementation: not only asking what could be built, but also how it could be made understandable and actionable for real users.
